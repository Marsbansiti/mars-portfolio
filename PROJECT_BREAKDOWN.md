# Mars Portfolio - Project Breakdown

## 📋 Project Overview

This is a personal portfolio website showcasing projects, skills, and professional experience. The portfolio is built primarily with HTML with minimal supporting technologies.

---

## 📁 Project Structure

```
mars-portfolio/
├── LICENSE                 # Strict proprietary license
├── README.md              # Project documentation
├── PROJECT_BREAKDOWN.md   # This file - detailed project breakdown
├── index.html             # Main portfolio landing page
├── about.html             # About me section
├── projects.html          # Projects showcase page
├── contact.html           # Contact/Get in touch page
├── assets/                # Static assets directory
│   ├── css/              # Stylesheets
│   │   ├── style.css     # Main stylesheet
│   │   └── responsive.css # Mobile responsive styles

│   │   └── utils.js      # Utility functions
│   ├── images/           # Image assets
│   │   ├── profile.jpg   # Profile picture
│   │   ├── projects/     # Project screenshots
│   │   └── icons/        # UI icons
│   └── fonts/            # Custom fonts (if any)

    └── metadata.json     # Site metadata
```

## 🎯 Key Features

### 1. **Landing Page (index.html)**
   - Hero section with introduction
   - Call-to-action buttons
   - Quick navigation to main sections
   - Featured projects preview

### 2. **About Section (about.html)**
   - Professional biography
   - Skills and competencies
   - Education and certifications
   - Timeline of experience

### 3. **Projects Showcase (projects.html)**
   - Detailed project cards
   - Project descriptions
   - Technologies used per project
   - Links to live demos and repositories
   - Filter/search functionality

### 4. **Contact Page (contact.html)**
   - Contact form
   - Social media links
   - Email contact information
   - Location details
